6. Najbardziej optymalne umieszczenie skryptu na stronie

  1. Aby umiścić skrypt w języku HTML korzytamy z tabu <scirpt>. Skrypt możemy umieeścić w każdym miejscu pliku HTML również wewnątrz nagłówka head. Pomiędzy tymi tagami podajemy zawartość skryptu. Domyślnym językiem jest Java Script. Kiedyś trzebya było podawać wartośc atrybuty type, w którym określano jaki to jest język. Teraz już nie trzeba tego robić.
  2. Wywołajmy najprostrzy skrypt: okieno allert które wypisuje test: alert("test"); To jest najprostrze wywołanie skrytpu, zatem skrypt możwmy uruchamiać wewnątrz języka HTML
  3. Trzeba pamiętać o średnikach na końcu linie w JS.
  4. skrypt zewnętrzny: <script src="js/nazwa-skrytpu.js"> Ścieżką może być również do zewnętrznego serwera, może to być jakaś witryna ze skryptem. Jednak gyd w momencie wczytywania skryput, padł server to wczytywanie strony zaiweszało się. Przed HTML 5 ten tag umieszczało się na końcu strony HTML.
  5. Strona www.caniuse.com i wpiszyemy w wyszukiwanie async i u stawimy kraj jako Polska tam mamy liczbe uzywanych przegladarek w Polsce i ich wersji. Zatem dodaje sie atrybut async, czyli asynchronicznie. Zatem zmieniamy domyślne zachownie przeglądrki.
